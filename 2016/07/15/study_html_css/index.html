选择器

标签选择器

类选择器

  • .开始
  • 字母，数字，-横线，下划线
  • 必须以字母开头
  • 区分大小写
  • 可以出现多次

id选择器

  • #开始
  • 字母，数字，-横线，下划线
  • 必须以字母开头
  • 区分大小写
  • 只能出现一次

通配符选择器

通配符*号：选中所有元素

属性选择器

  • [disabled]、[type=”button”]…
  • [class~=sports]：class中包含了sports的元素
  • [lang|=en]：lang属性，以en或en-中划线开头
  • [href^=”#”]：href中以#开头的
  • [href$=pdf]：href中以pdf结尾的
  • [href*=”lady.163.com”]：href中包含了lady.163.com的

伪类选择器

伪类选择器 描述
a:link 选中所有的链接（只能用在a标签）
a:visited 访问过的链接（只能用在a标签）
:hover 鼠标经过
:active 点击时
伪类选择器 描述
input:enabled 可用的状态
input:disabled 禁用的状态
input:checked 选中的
伪类选择器 描述
li:first-child 第一个li
li:last-child 最后一个li
li:nth-child(even/odd/3n+1) 奇偶行/n=0，3n+1等于1，4，7…
li:nth-last-child(even/odd/3n+1) 倒数
:only-child 只有一个子元素的
dd:first-of-type 第一个dd类型的
dt:last-of-type 最后一个dt类型的
dd:nth-of-type(even) 偶数个dd
dt:nth-last-of-type(2n) 倒数偶数项的dt
span:only-of-type 唯一一个span子元素
伪类选择器 描述
:empty 没有子元素（包括文本节点）的每个匹配元素
:root 选中根标签html
:not(selector) 选择器不是selector的所有元素
:target 突出锚点对应的内容
:lang(language) 制定语言的匹配元素

伪元素选择器

伪元素选择器 描述

::before 需要配合content:””，content中的值就是插入的内容
::after 需要配合content:””，content中的值就是插入的内容

选择器的浏览器支持情况

文本

简写：font:font-style可选 font-weight可选 font-size /line-height可选 font-family;

font-size

  • em和%是相对于父元素的字号

font-style

  • normal
  • italic：使用字体自带的斜体
  • oblique：字体没有斜体时，强制倾斜

line-height

  • normal：通常1.1~1.4
  • line-height:3和line-height:300%的区别：300%先计算再继承，所以子元素的字号变化不会影响行高，3反之

white-space

  • normal：浏览器决定，一般为换行、多个空格和tab会被合并为空格，可以自动换行
  • nowrap：不换行，强制一行
  • pre：保留换行，多个空格和tab保留，不会自动换行
  • pre-wrap：pre的基础上允许自动换行，常用于写代码块
  • pre-line：只保留换行，多个空格和tab合并为空格，会自动换行

word-wrap

  • normal
  • break-word：长单词允许自动换行

word-break

  • normal
  • keep-all：不做打断
  • break-all：所有的都可打断自动换行

text-shadow

  • none
  • x轴偏移 y轴偏移 模糊半径 颜色

text-decoration

  • underline overline line-through
  • 可写多个

text-overflow

  • clip默认值
  • ellipsis

cursor鼠标指针

  • zoom-in：放大镜
  • zoom-out：缩小镜

强制继承 inherit

inherit：文本类型的css几乎都有这个值。

盒模型

margin

相邻的元素，外边距会合并，取较大值。

父元素与第一个/最后一个子元素，外边距也会合并。

border-radius

box-shadow

  • box-shadow: inset为内阴影 水平偏移 垂直偏移 模糊半径非必要 阴影大小非必要 颜色;
  • 颜色不设置时，默认为文字颜色
  • 多阴影用逗号分隔

outline

  • 不占空间
  • 在border外

背景

background-image

  • 多个背景图时，先写的在最上边，后写的在下边
  • 有背景图的基础上，写background-color时，bgc永远处于最下面一层

background-repeat

  • 两个值时对应x和y轴是否平铺，可以对应多张背景图
  • 多个值时，每个值对应的是每一张背景图

background-attachment（可做视察）

  • 可以写多个值，和图片一一对应
  • 默认值scroll，背景图不动
  • local，背景会随着内容一起动
  • fixed，尝试做视差滚动。

background-position

  • background-position:20% 50%; 代表图片的x轴20%和y50%的点，定位在容器的x20%y50%的位置
  • 一个值时，另外一个被忽略的值默认为center
  • background-position:right 10px top 20px;right和top相当于参照的点
  • 配合雪碧图

linear-gradient()线性渐变

radial-gradient()径向渐变

repeat-*-gradient重复渐变

  • background-image: repeating-linear-gradient(red,blue 20px, red 40px);

background-origin 规定背景区域

  • 类似box-sizing
  • border-box
  • padding-box
  • content-box

background-clip 裁剪

  • border-box
  • padding-box
  • content-box
  • 类似background-origin，互相配合使用

background-size

  • cover，撑满容器
  • contain，宽高不超出容器
  • 数值/100%

background简写

  • background: bg-image bg-position / bg-size bg-repeat bg-attachment box-origin box-clip color;
  • box-origin和box-clip只写了一个的时候，这个值对应两个属性
  • 多个用逗号分隔

布局

布局 - 元素的摆放模式

display

设置元素的显示方式

display 默认宽度 可设置宽高 起始位置
block 父元素宽度 是 换行
inline 内容宽度 否 同行，内容换行
inline-block 内容宽度 是 同行，整体换行
none 设置元素不显示

z-index栈

position

relative
  • 在文档流中
  • 以自身为参照物
absolute
  • 默认宽度为内容宽度
  • 脱离文档流
  • 参照物是第一个有定位的祖先 或 根元素
fixed
  • 默认宽度为内容宽度
  • 脱离文档流
  • 参照物为视窗

三行自适应布局

  • 顶栏固定在上方
  • 低栏固定在下方
  • 内容区自适应

float

  • 默认宽度为内容宽度
  • float的元素在同一文档流中
  • float的元素半脱离文档流
  • 对元素，脱离文档流；对内容，在文档流

clear

  • 应用于浮动元素的后续元素
  • 应用于块级元素

flex

  • 弹性布局的元素：flex container（弹性的容器）
  • 弹性的子元素：flex item
  • flex排列的方向：主轴main axis
  • 和主轴垂直的方向：副轴 cross axis
创建flex container

display: flex;

flex item
  • 在文档流中的子元素才是flex item
  • 子元素才是！孙元素不是！
方向
  • flex-direction：弹性的方向
  • flex-wrap：弹性的换行
  • flex-flow：弹性的流
  • order：顺序
flex-direction
column column column column column
弹性的方向 行：左至右 反向行：右至左 列：上至下 反向列：下至上
flex-direction row row-reverse column column-reverse
flex-wrap
column column column column
弹性换行 不换行 换行 反向换行
flex-wrap nowrap wrap wrap-reverse
flex-flow

值缩写，同时设置flex-direction和flex-wrap

语法：

  • flex-flow: <”flex-direction”> || <”flex-wrap”>
  • 可写一个值，也可都写
order
  • 值是整数，可以是负数
  • 初始值是0
  • 负值排在前面
  • order值大的元素，排在主轴的最后面
弹性
flex-basis
  • 设置flex item的初始宽/高
  • 值是main-size：row时，main-size相当于width；column时main-size相当于height
flex-grow
  • 将容器内多余的空间，分配给设置了flex-grow的flex item
  • 值是数字
  • 初始是0
  • 计算公式：flex-basis + flow-grow/sum(flow-grow) * remain
  • 宽或高 + 分配的grow份数/所有子元素总共的grow数 * 多余空间
flex-shrink
  • 值是整数
  • 初始值1
  • 当所有flex item的宽度超出flex container的宽度、也就是说container的宽度剩余为负数时，通过flex-shrink将超出的宽度分配给item
  • 计算公式：flex-basis + flow-grow/sum(flow-grow) * remain
  • 宽或高 + 分配的shrink份数/所有子元素总共的shrink数 * 多余空间
flex值缩写
  • flex: flex-grow || flex-shrink || flex-basis
  • 默认值分别为：0 1 main-size
  • 可写一个可全写
对齐
justify-content
  • 设置主轴main-axis方向上的对齐方式
  • 可以理解为：用来设置如何分配剩余空间
  • 给容器设置，容器内所有item都生效
column column column column column column
对齐方式 主轴的起点 主轴的终点 居中 item的间隔平分了空白空间 包括首尾也平分空白空间
justify-content flex-start flex-end center space-between space-around
align-items
  • 设置副轴cross-axis方向上的对齐方式
  • 给容器设置，容器内所有item都生效
column column column column column column
对齐方式 主轴的起点 主轴的终点 居中 基线对齐 延展充满副轴空间
align-items flex-start flex-end center baseline stretch
align-self
  • 设置单个flex item在副轴cross-axis方向上的对齐方式
column column column column column column column
对齐方式 使用容器设置的对齐方式 主轴的起点 主轴的终点 居中 基线对齐 延展充满副轴空间
align-self auto flex-start flex-end center baseline stretch
align-content
  • 设置副轴cross-axis方向上，行的对齐方式
column column column column column column column
对齐方式 主轴的起点 主轴的终点 居中 每一行的间隔平分了空白空间 包括首尾也平分空白空间 延展充满
align-content flex-start flex-end center space-between space-around stretch

变形

transform

变形 值 值
transform none transform:rotate() translate() scale() skew()
  • 值的先后顺序不同，元素的位置也就不同
transform: rotate(deg)旋转
  • 正值顺时针旋转
  • 负值逆时针旋转
transform: rotate3d(deg)旋转
  • rotate3d(x, y, z, deg)
transform: translate()移动
移动 值
translate( x,y)
translateX( 偏移度)
translateY( 偏移度)
transform: translate3d()移动
  • 多一个z轴参数
  • translateZ(length)
transform: scale()缩放
缩放 值
scale( x,y)第二个值不写时默认等于第一个值
scaleX( 缩放倍数)
scaleY( 缩放倍数)
transform: scale3d()缩放
  • scaleZ(倍数)
  • 可以使translateZ的值放大多倍
transform: skew(deg)倾斜
倾斜 值
skew( y往x倾斜,x往y倾斜)第二个值不写时默认等于第一个值
skewX( y往x的倾斜度)
skewY( x往y的倾斜度)

属性 值

perspective透视效果

perspective-origin透视的角度

perspective-origin x轴：center/left/right/%/px y轴：center/top/bottom/%/px

step-end

  • 设置延时多久再开始动画
属性 值 值 值
transition-delay 默认值：0s 1s 1s, 2s, …

transition

  • none / transition-property transition-duration transition-timing-function transition-delay
