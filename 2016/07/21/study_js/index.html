逻辑与
  • 取值：如果第一个操作数转换为布尔值为true的话，那么它的值就是第二个操作数

.length

.charAt(index)

  • 获取索引值对应的字符

.indexOf(searchValue)

  • 返回searchValue的索引值
  • 如果searchValue是一个字符串，则返回该字符串的起始位置
  • 当searchValue不存在时，返回-1

.search(regexp)

  • 参数可以是正则
  • 搜索的值不存在时，返回-1

.match(regexp)

  • 匹配
  • 返回数组
  • 匹配不到时，返回null

.replace(regexp|str, newStr)

  • 替换

.substring(index, index)

  • 从第一个索引值开始，到第二个索引值结束，截取字符串
  • 第一个索引值是被包含的，第二个索引值不被包含
  • 只写一个索引值作为参数时，这个索引值后面所有的字符都被一起截取出来

.slice(begin, end)

  • 从第一个索引值开始，到第二个索引值结束，截取字符串
  • 第一个索引值是被包含的，第二个索引值不被包含
  • 只写一个索引值作为参数时，这个索引值后面所有的字符都被一起截取出来
  • 可以传入负值，负值代表倒数

.substr(start, length)

  • 从某个索引开始，取指定长度的字符串
  • 第二个参数不写时，取到最后

数组

  • reverse、sort、push、unshift、shift、pop、splice这些方法都改变了原数组
  • slice、concat、join、map、reduce这些方法没有修改原数组

创建数组

  • var arr = new Array();
  • var arr = [];

arr.length

  • 获取数组的长度

获取数组的元素

  • 通过索引值获取

修改数组的元素

arr.indexOf(search)

  • 查询数组里的元素
  • 如果有，返回索引值
  • 如果没有，返回-1

arr.forEach(callback)

  • 每遍历到一个元素，就会执行传入的方法，并且把遍历到的数据，传给方法
  • 对callback的回调参数是有要求的：必须是3个值，第一个表示遍历到的元素，第二个表示索引，第三个值表示数组本身

arr.reverse()

  • 把数组的元素顺序颠倒

arr.sort(回调函数)

  • 排序
  • 排序后会直接改变原来的数组
  • 如果不传回调函数，则按照unicode排序

arr.push()

  • 在已有数组的基础上，再加上一些元素
  • 可以一次性增加多个，用逗号分隔
  • 往后添加

arr.unshift()

  • 和push()基本类似
  • 往数组的前面添加

arr.shift()

  • 获取数组内的第一个元素，并且原数组会删掉这个元素

arr.pop()

  • 获取数组内的最后一个元素，并且原数组会删除掉这个元素

arr.splice(index，howMany，elements)

  • 在数组的任意位置插入或删除
  • index：从哪个位置开始删除
  • howMany：删除的数量,如果是0，则在index的位置插入新的elements
  • elements：可选，新添加的元素们（单个/多个），从index开始添加

arr.slice(begin，end)

  • 拷贝：从begin索引位置开始，到end索引位置结束
  • index包含在拷贝中
  • end可选，不包含在拷贝中，不传end时，则拷贝到最后
  • 不改变原数组

arr.concat(valueN)

  • 连接，可连接多个值
  • value可以是非数组的数据类型

arr.join(分隔符)

  • 通过指定的分隔符，将数组组成字符串
  • 不填写指定分隔符，将默认用逗号分隔
  • 如果填写空的字符串作为分隔符，则没有任何分隔符做连接

案例a：保留原有数组、修改数组内元素并生成新的数组

arr.map(callback)

  • 遍历，将callback的所有返回都加到新的数组里面

arr.reduce(callback,[initialValue])

  • 类似于：1+2+3+4的运算过程

  • 匹配一个位置

  • $：结尾位置/\.jpg$/.jpg结尾的
  • \b：单词边界/\bis\b/is必须是一个完整的单词

字符类

  • 匹配一类字符中的一个
  • [abc]：a或b或c
  • [0-9]：范围内的一个数字
  • ^在方括号内时表示非xxx的字符：[^0-9]非数字的一个字符
  • [a-z]：一个字母
  • .：任意字符（换行除外）

元字符

  • 具有特殊意义的字符
  • ^、$、\b
  • \d：等价于[0-9]
  • \D：等价于[^0-9]，非数字
  • \s：空白符
  • \S：非空白符[^\s]
  • \w：[A-Za-z0-9_]
  • \W:[^\w]

量词

  • 表示出现的次数
  • {m,n}：表示m到n次之间
  • *：{0,}0到任意次数
  • ?：{0,1}0次或1次
  • +：{1,}1次或任意次数

转义符

  • 需要匹配的字符是元字符

  • 或

  • (?:)：不捕获
