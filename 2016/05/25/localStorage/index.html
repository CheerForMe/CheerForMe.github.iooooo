LocalStorage基础操作

发表于 2016-05-25   |   分类于 LocalStorage

web storage

让应用在用户本地存储内容，这些内容是成对出现的：内容名称，内容值。

web storage分为两个部分：

  • Local Storage：本地存储空间，存储的数据在关掉浏览器后不会消失。
  • Session Storage：连线时段存储空间，关掉浏览器会自动清除。

storage事件

当存储区域发生变化时，会触发storage事件：设置、移除、清空数据时，都会触发事件。

数据触发的当前窗口不会发生这个事件，在其他窗口会发生。前提是其他窗口可以访问到发生变化的窗口。

sessionStorage：同一个窗口里的框架发生变化时，才会触发storage事件。

localStorage作用域

作用域/有效的应用范围：

  • 取决于文档的来源
  • 同一个文档来源可以互相使用localStorage数据
  • 来源取决于：主机名、端口号、协议
